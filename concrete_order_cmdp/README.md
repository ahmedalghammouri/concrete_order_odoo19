# Concrete Mixer Delivery Point (CMDP)

Fast point interface for concrete mixer delivery operations.

## Features

- Quick access to pending delivery tickets
- Barcode scanning for fast retrieval
- Large buttons and displays for operators
- Real-time delivery status updates
- Session management for operators

## Installation

1. Install dependencies: concrete_order, inventory_scale_integration_weighpoint
2. Update module list
3. Install concrete_order_cmdp

## Usage

Access via: Concrete Delivery Point > Delivery Point Interface
